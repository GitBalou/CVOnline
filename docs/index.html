Mini RPG

Réalisé en C il y a bien longtemps, le but était avant tout récréatif!

  • A partir d'un fichier texte décrivant une carte, le logiciel met en mémoire les objets composant un monde imaginaire.
  • Un service dédié gére les animations de l'environnement (eau qui coule etc...) et des personnages. Il déplace également les PNJ selon un trajet prédéfini dans un autre fichier texte.
  • Le déplacement du personnage principal est réalisé suivant les commandes entrées par le joueur, une carte "cachée" ("solids") permet de gérer les collisions avec le décor.
    L'interaction avec des PNJ est également possible à travers des dialogues.
  • Un moteur graphique simple affiche le rendu du monde dans son état actuel (positions des personnages, dialogue éventuel, ...).
  • Les graphismes sont tirés de tileset & sprites issus de rpg maker, trouvés à l'époque sur internet.

Réalisé en C sous l'IDE codeblock, en s'appuyant notamment sur la librairie SDL.
Année de réalisation: 2014

Sea.me

Naviguant régulièrement en mer l'envie m'a pris d'enregistrer mes voyages en direct afin de pouvoir revivre mes navigations et des les partager.

  • L'application repère la position de l'utilisateur via le GPS. L'enregistrement de la trace gps démarre automatiquement une fois en mer.
  • Il est possible de noter des évènements particulier de la navigation : vent, pression, mouillage, photos .... Ces évènements sont géoréféréncés et représentés sur la carte.
  • Il est possible de définir un itinéraire à l'avance, et ainsi de récupérer les marées aux ports de départ et d'arrivée, ainsi que la météo du parcours. Les données sont récupérées via des API gratuites sur le web (open weather map ...)
  • Lorsque du réseau GSM est disponible, les données de navigation sont transmises en temps réel sur un serveur distant pour permettre un accès aux données depuis différentes plateformes (application mobile / tablette / site web).

Application mobile hybride fonctionnant sur Android & Ios.
Réalisée sur le framework Ionic :cordova (Phonegap) + Angular Js.
Partie serveur réalisée en php / Mysql.
Année de réalisation: 2015.
Actuellement en test alpha sous Android

Geoviti

Réalisée à la demande d'un ami viticulteur, Geoviti est une application de gestion embarquée d'un vignoble.

  • L'application s'appuie sur une base de donnée contenant notamment les informations parcellaires, les opérations à réaliser au champ etc...
  • Le vigneron peut voir son parcellaire sous différents angles : cépages présent, rendement obtenu, photographies, etc...
  • Il est également possible de naviguer dans son vignoble à travers le temps, et ainsi de revoir les rendement des années précédentes par exemple.
  • L'application utilise le GPS pour suivre le déplacement de l'utilisateur et repérer sa position dans les parcelles. Si une opération culturale est en cours (ex: labour des vignes), l'application retransmet au serveur la date et l'heure,ainsi que les donneés sur la parcelle et l'opération réalisée. La traçabilité du vignoble se remplie ainsi en temps réel.

Application mobile hybride fonctionnant sur Android et Ios.
Réalisée sur le framework Ionic :cordova (Phonegap) + Angular Js.
Partie serveur réalisée en php / Mysql.
Année de réalisation: 2016.
Actuellement en développement.
